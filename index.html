Why does this website look so dated?

This website uses just HTML, which means there isn't any style, colors, or fancy functionality. Even though it looks like a website from the 90s, this website was coded it up on June 18th, 2022. (Nothing significant about the date, it's just when I took the time to do it).

What is "HTML", anyway?

The techy definition: HTML stands for "Hypertext Markup Language". This website file is stored on a server somewhere, and when you clicked on the link to get here, that server sent the files to your browser (hopefully Chrome).
